CODE CRACKER

Think you've got what it takes to crack the code?

HOW TO PLAY

  1. Decode the hidden code: As the codebreaker, your mission is to decipher the secret code.
  2. Guess the code: You have a total of twelve turns to crack the code's order and colours. Each turn, you select a row of code pins on the board to make your guess.
  3. Get feedback: After each guess, you'll receive feedback in the four pins next to your guess row. A black key pin means you've guessed a code pin in the correct colour and position. A white key pin indicates you've got the right colour, but it's in the wrong spot.
  4. Keep guessing: The game continues with guesses and feedback until you either solve the code or fill up all the rows on the board, and it’s game over!
  5. Rack up points: For every correct code you crack, you earn 100 points. But there's an extra challenge – you’ll earn an additional 100 points for every unused row on the board.

Are You Up to the Challenge?: Can you break the code and earn the highest score? Get ready for an exciting code-cracking adventure!
